# Command Bar 变更文件列表展开位置不符合预期

## 现象

- Chat 底部 Command Bar 显示 “N files with changes” 时，点击展开的文件 diff 列表出现在**摘要栏上方**。
- 与 Cursor 等产品的交互习惯不符：期望列表在 “N files with changes” **下方**展开。

## 原因

`SidebarChat.tsx` 中 Command Bar 的 DOM 结构将可折叠文件列表组件放在 summary bar **之前**渲染，CSS flex 顺序导致视觉上列表出现在上方。

## 修改方案

调整 `SidebarChat.tsx` 中 Command Bar 布局：

1. 先渲染 summary bar（“N files with changes” + 操作按钮）。
2. 再在 summary bar **下方**条件渲染展开的文件列表。

修改后需执行 `npm run buildreact` 打包 React 资源。

**涉及文件**：`browser/react/src/sidebar-tsx/SidebarChat.tsx`
